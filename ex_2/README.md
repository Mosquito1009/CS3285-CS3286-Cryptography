## 幻方密码加密解密

幻方密码相较于置换密码，需要对密钥进行取逆。

运行结果如下所示：

```log
请输入加解密内容：4D 61 65 53 2D 4F 69 20 4E
请输入加密密钥：8,1,6,3,5,7,4,9,2
是否解密内容（Y/n）：n
是否幻方加密（Y/n）：Y
结果: 61 4E 53 69 2D 65 4F 4D 20

请输入加解密内容：61 4E 53 69 2D 65 4F 4D 20
请输入加密密钥：8,1,6,3,5,7,4,9,2
是否解密内容（Y/n）：Y
是否幻方加密（Y/n）：Y
结果: 4D 61 65 53 2D 4F 69 20 4E

```
